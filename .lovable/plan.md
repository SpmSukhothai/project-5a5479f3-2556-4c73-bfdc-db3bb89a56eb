## เป้าหมาย

ปรับฟอร์มทุกหน้าที่เกี่ยวข้อง (ตั้งค่าอัตรา `/settings`, ลงทะเบียนบุตร, บันทึกการเบิก) ให้แสดง/ซ่อนตัวเลือก "เงินอุดหนุน" และ "กลุ่มสาขาวิชา" ตามเงื่อนไขจริงของระเบียบ และแยกกลุ่มสาขาของ ปวช. (8 สาขา) กับ ปวส. (2 กลุ่ม)

## เงื่อนไขการแสดงผล (ใช้กับทุกฟอร์ม)

```text
ประเภทโรงเรียน = ราชการ
  → ไม่แสดง "เงินอุดหนุน"  (บังคับค่าเป็น none)

ประเภทโรงเรียน = เอกชน  →  เลือกระดับชั้นก่อน
  ├─ อนุบาล / ประถม / ม.ต้น / ม.ปลาย / ปวช.
  │     → แสดง "เงินอุดหนุน" เฉพาะ [รับเงินอุดหนุน / ไม่รับเงินอุดหนุน]
  │       (ไม่มีตัวเลือก none)
  │     → ถ้าเป็น ปวช. : แสดงกลุ่มสาขา 8 สาขา
  └─ ปวส.
        → ไม่แสดง "เงินอุดหนุน"  (บังคับค่าเป็น none)
        → แสดงกลุ่มสาขา 2 กลุ่ม
```

## กลุ่มสาขาวิชา

**ปวช. — 8 สาขา (มีอยู่แล้วในฐานข้อมูล):** ช่างอุตสาหกรรม, พาณิชยกรรม, ศิลปกรรม, คหกรรม, เกษตรกรรม, ประมง, ท่องเที่ยว, สิ่งทอ

**ปวส. — 2 กลุ่ม (เพิ่มใหม่):**
- กลุ่มที่ 1 — ช่างอุตสาหกรรม/อุตสาหกรรม, เทคโนโลยีสารสนเทศและการสื่อสาร, ทัศนศาสตร์
- กลุ่มที่ 2 — พาณิชยกรรม/บริหารธุรกิจ, ศิลปกรรม/ศิลปหัตถกรรม, เกษตรกรรม, คหกรรม, อุตสาหกรรมการท่องเที่ยว

## รายละเอียดทางเทคนิค

### 1. ฐานข้อมูล (migration)
- เพิ่มคอลัมน์ `level` (text) ในตาราง `program_groups` เพื่อผูกกลุ่มสาขากับระดับ
- อัปเดต 8 สาขาเดิม → `level = 'vocational_certificate'` (ปวช.)
- เพิ่ม 2 แถวใหม่ → `level = 'higher_vocational'` (ปวส.):
  - `code = 'voc_hi_group1'`, name = "กลุ่มที่ 1 (ช่างอุตสาหกรรม/เทคโนโลยีสารสนเทศและการสื่อสาร/ทัศนศาสตร์)"
  - `code = 'voc_hi_group2'`, name = "กลุ่มที่ 2 (พาณิชยกรรม/บริหารธุรกิจ/ศิลปกรรม/เกษตรกรรม/คหกรรม/ท่องเที่ยว)"

### 2. `src/lib/labels.ts`
- เพิ่ม `PRIVATE_SUBSIDY_TYPES = ['subsidized', 'non_subsidized']`
- เพิ่ม helper `showsSubsidy(school_type, education_level)` → true เฉพาะ เอกชน และ ไม่ใช่ ปวส.
- ปรับ `findRate` ไม่ต้องแก้ (คีย์ครบ 5 ตัวรองรับอยู่แล้ว)

### 3. ฟอร์มทั้ง 3 หน้า (`settings.tsx`, `children.tsx`, `reimbursements.tsx`)
- โหลด `program_groups` พร้อมคอลัมน์ `level`; กรองตัวเลือกกลุ่มสาขาตามระดับที่เลือก (ปวช.=8, ปวส.=2)
- ซ่อน/แสดงช่อง "เงินอุดหนุน" ตามเงื่อนไข; เมื่อซ่อนให้ set `subsidy_type = 'none'` อัตโนมัติ
- เมื่อแสดง ให้มีเฉพาะ `subsidized` / `non_subsidized` และถ้าค่าเดิมเป็น none ให้ default เป็น `subsidized`
- เมื่อเปลี่ยนระดับชั้น ให้ล้าง `program_group_id` หากกลุ่มเดิมไม่อยู่ในระดับใหม่

### ผลกระทบ / ความเข้ากันได้
- ข้อมูลอัตรา/บุตรเดิมยังคงอยู่ (กลุ่มสาขาเดิมถูกผูกเป็นระดับ ปวช.)
- trigger `validate_program_group` ที่บังคับให้ ปวช./ปวส. ต้องมีกลุ่มสาขา ยังทำงานเหมือนเดิม
