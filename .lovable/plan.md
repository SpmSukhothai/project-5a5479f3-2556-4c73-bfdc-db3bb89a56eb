## ปรับหน้า /schools และฐานข้อมูลโรงเรียนต้นสังกัด

### บริบท
ตาราง `schools` คือฐานข้อมูล "โรงเรียนต้นสังกัด" ของผู้มีสิทธิ (อ้างอิงจาก `guardians.school_id`)
โรงเรียนที่บุตรกำลังศึกษาเก็บแยกเป็นข้อความ (`study_place`) ไม่เกี่ยวกับตารางนี้ — การเปลี่ยนตาราง `schools` จึงไม่กระทบหน้าบุตร/เบิกจ่าย

### 1. จำกัดสิทธิ์เข้าหน้า /schools เฉพาะ admin
- `src/routes/_authenticated/schools.tsx`: เพิ่มการตรวจสิทธิ์ — ถ้าไม่ใช่ admin ให้พาไปหน้า dashboard (รูปแบบเดียวกับหน้า settings: `if (role && role !== "admin") return <Navigate to="/dashboard" />;`)
- `src/components/AppShell.tsx`: เพิ่ม `adminOnly: true` ให้เมนู `/schools` เพื่อซ่อนจาก sidebar ของผู้ที่ไม่ใช่ admin

### 2. เอาคอลัมน์ "ประเภท" และ "จังหวัด" ออกจากหน้า /schools
- ลบหัวตารางและช่องข้อมูล "ประเภท" และ "จังหวัด" ในตารางรายการ
- ลบช่องกรอก "ประเภท" และ "จังหวัด" ในกล่องเพิ่ม/แก้ไขโรงเรียน
- ยังคงเก็บค่าในฐานข้อมูลเป็นค่าเริ่มต้นอัตโนมัติ (ประเภท = ราชการ, จังหวัด = สุโขทัย) เพราะคอลัมน์ในฐานข้อมูลห้ามว่าง — ไม่ลบคอลัมน์ในฐานข้อมูล จึงไม่กระทบหน้าอื่น

### 3. ตั้งฐานข้อมูลให้มีเฉพาะ 27 โรงเรียน
- ลบโรงเรียนเอกชน 2 แห่งที่ไม่อยู่ในรายการ (กวงฮั้ว, สารสาสน์สุโขทัย) — ไม่มีผู้มีสิทธิอ้างอิง จึงลบได้ปลอดภัย
- คงโรงเรียน 5 แห่งที่ตรงกับรายการอยู่แล้ว (อุดมดรุณี, บ้านสวนวิทยาคม, บ้านด่านลานหอยวิทยา, คีรีมาศพิทยาคม, ศรีสำโรงชนูปถัมภ์) — เพื่อรักษาการอ้างอิงของผู้มีสิทธิเดิม
- เพิ่มอีก 22 โรงเรียนที่เหลือ โดยเติมคำว่า "โรงเรียน" นำหน้าทุกชื่อ ตั้งประเภท = ราชการ, จังหวัด = สุโขทัย และสร้างรหัสโรงเรียนอัตโนมัติ

รายชื่อ 22 โรงเรียนที่เพิ่ม: สุโขทัยวิทยาคม, ยางซ้ายพิทยาคม, ลิไทพิทยาคม, กงไกรลาศวิทยา, หนองตูมวิทยา, ไกรในวิทยาคม รัชมังคลาภิเษก, ตลิ่งชันวิทยานุสรณ์, บ้านใหม่เจริญผลพิทยาคม, ชัยมงคลพิทยา, ศรีนคร, ท่าชัยวิทยา, บ้านแก่งวิทยา, เมืองด้งวิทยา, บ้านไร่พิทยาคม, วังทองวิทยา, ขุนไกรพิทยาคม, สวรรค์อนันต์วิทยา, หนองปลาหมอวิทยาคม, หนองกลับวิทยาคม, สวรรค์อนันต์วิทยา 2, ทุ่งเสลี่ยมชนูปถัมภ์, เมืองเชลียง

### ผลลัพธ์
- เฉพาะ admin เห็นและเข้าหน้า /schools ได้
- หน้า /schools แสดงเฉพาะ ลำดับ / รหัสโรงเรียน / ชื่อโรงเรียน / จัดการ
- ฐานข้อมูลโรงเรียนต้นสังกัดมีครบ 27 โรงเรียนตามรายการ ไม่กระทบหน้า /guardians, /children, /reimbursements
